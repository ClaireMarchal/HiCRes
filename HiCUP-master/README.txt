HiCUP (Hi-C User Pipeline)
--------------------------

HiCUP is a bioinformatics pipeline for processing Hi-C data. 

For full instructions on how to use HiCUP please read the markdown files (text format) in the 'Documentation' folder (we suggest you begin by reading the 'overview.md' file).  Full documentation is also provided on the HiCUP homepage at: 

http://www.bioinformatics.babraham.ac.uk/projects/hicup

Primary contact: steven.wingett@babraham.ac.uk
